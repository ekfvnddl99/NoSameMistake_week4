# 되어줘, 명예 곰돌이!
------------
'인간은 똑같은 실수를 반복한다.'  
라는 말에서 아이디어를 얻어 만들어진 실수 방지 앱입니다.  
현재-미래-과거 시간 순서대로 만들어진 탭에서 여러분들의 실수를 관리해보세요!  
다른 유저들의 실수를 보고싶은 기준에 맞게 정렬하여 볼 수 있습니다.  

src/screens에 들어가면 각 탭과 관련된 코드를 볼 수 있습니다.
